# 디지털 시계 🕐

현재 시간을 시·분·초로 깔끔하게 보여주는 디지털 시계 웹사이트입니다.

## 기능

- ⏰ **실시간 디지털 시계** — 시·분·초(HH:MM:SS) 표시
- 🔠 **글자 크기 조절** — `A−` / `A+` 버튼으로 시계 크기를 키우거나 줄임 (40%~250%)
- 📅 **날짜 표시/숨김** — `📅` 버튼으로 날짜·요일 표시를 켜고 끔
- 🕛 **12 / 24시간제 전환** — `24` ↔ `12` 버튼으로 전환 (24시간제에서는 AM/PM 숨김)
- 🅰 **폰트 변경** — 버튼을 눌러 4가지 폰트를 순환 (기본 · Orbitron · Roboto Mono · 손글씨)
- 🌙 **라이트 / 다크 모드** — 우측 상단 버튼으로 전환, 선택한 테마는 자동 저장
  (첫 방문 시 시스템 설정을 따름)
- ⛶ **전체화면 모드** — 버튼 또는 키보드 `F` 키로 토글
- 📱 **반응형 디자인** — 화면 크기에 맞춰 글자 크기 자동 조절

> 글자 크기, 날짜 표시, 12/24시간제, 폰트, 테마 선택은 모두 브라우저에 저장되어 다음 방문 때도 유지됩니다.

## 사용법

`index.html` 파일을 브라우저에서 열기만 하면 됩니다. 별도의 빌드나 의존성이 없습니다.

```bash
# 간단히 로컬 서버로 띄우려면
python3 -m http.server
# 브라우저에서 http://localhost:8000 접속
```

## 파일 구성

| 파일 | 설명 |
| --- | --- |
| `index.html` | 페이지 구조 |
| `style.css` | 스타일 및 라이트/다크 테마 |
| `script.js` | 시계 동작, 테마 전환, 전체화면 로직 |
| `.github/workflows/deploy.yml` | GitHub Pages 자동 배포 워크플로 |

## 배포 (GitHub Pages)

`main` 브랜치에 푸시하면 워크플로가 자동으로 사이트를 배포합니다.
처음 한 번만 아래 설정이 필요합니다.

1. GitHub 저장소 → **Settings → Pages** 이동
2. **Source** 를 **"GitHub Actions"** 로 선택
3. 이후 `main` 에 푸시될 때마다 자동 배포 (Actions 탭에서 수동 실행도 가능)

배포가 끝나면 아래 주소로 접속할 수 있습니다.

